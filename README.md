üß† Methodology
Dataset: Elliptic Bitcoin Dataset (203K+ transactions, 166 features, 234K edges).
Feature Mapping: Expands features using sinusoidal/Fourier transforms.
Model: 2-layer GAT with attention heads, trained on quantum-enhanced features.
Evaluation: Confusion matrix, ROC/AUC, temporal robustness, and forensic explanations.
Pipeline: Modular scripts and notebooks for setup, training, evaluation, and explanation.
